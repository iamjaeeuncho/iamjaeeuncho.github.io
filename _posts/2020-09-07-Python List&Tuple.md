## 숫자형 타입 (numbers)
- 정수, 실수로 구성
- 수학의 기본 연산자(가감승제) 사용 가능

```python
a = 5
b = 4

print(a + b)
print(a * b) # 곱하기
print(a - b) 
print(a / b)
print(a % b) # 나머지
print(a ** b) # 지수승
```
