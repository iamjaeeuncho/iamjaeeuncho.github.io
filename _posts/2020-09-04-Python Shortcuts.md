일단 단축키는 'HELP' 메뉴에서 'Keyboard shortcuts'를 누르거나  
키보드에서 H 버튼을 누르면 단축키 내용을 볼 수 있다  


![shortcuts](https://user-images.githubusercontent.com/46369038/109161610-9c9ba300-77ba-11eb-8c0a-dbe6392ac6cb.png)  


자주 쓰는 단축키는 아래 정도  

Y: 코드 모드로 변경  
M: 마크다운 모드로 변경  


Ctrl+Enter: 셀만 실행  
Shift+Enter: 셀 실행하고 아래 셀로 이동  
Alt+Enter: 셀 실행하고 아래 셀 추가  


A: 위에 셀 삽입  
B: 아래에 셀 삽입  
X: 셀 자르기  
C: 셀 복사하기  
D,D: 셀 삭제하기  


L: 줄 번호 표시  
ctrl+/: 주석처리  
shift+tab: 함수 상세 설명  
tab: 자동 완성
