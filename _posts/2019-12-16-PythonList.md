## 리스트(List)란?
* 복합 데이터의 집합
* 식별: 대괄호([ ])로 감싸 주고 각 요소값들은 쉼표(,)로 구분
* 리스트 인덱싱과 슬라이싱은 문자열과 동일하다. 인덱싱은 하나, 슬라이싱은 구간
* 연산: 더하기(+), 곱하기(*), 길이구하기(len함수)
* 리스트 값 수정은 그냥 덮어쓰기, 삭제는 del함수 또는 슬라이싱 활용


## 1. 리스트 만드는 법
* 대괄호([ ])로 감싸 주고 각 요소값들은 쉼표(,)로 구분
* 리스트 안에는 어떤 자료형도 포함 가능
* ※ 비어 있는 리스트는 a = list()로 생성할 수도 있다.
```python
>>> a = []                                 # 비어 있는 리스트
>>> b = [1, 2, 3]                          # 숫자를 요소값으로 가진 리스트
>>> c = ['Life', 'is', 'too', 'short']     # 문자열을 요소값으로 가진 리스트
>>> d = [1, 2, 'Life', 'is']               # 숫자, 문자을 같이 가진 리스트
>>> e = [1, 2, ['Life', 'is']]             # 리스트 안에 리스트
```


## 2. 리스트의 인덱싱
```python
# 리스트 인덱싱
>>> a = [1, 2, 3, ['a', 'b', 'c']]
>>> a[0]
1
>>> a[-1]
['a', 'b', 'c']                      # a[-1]은 마지막 요소값인 ['a', 'b', 'c']를 나타냄
>>> a[3]
['a', 'b', 'c']

# 리스트를 포함한 2중 리스트의 인덱싱
>>> a = [1, 2, 3, ['a', 'b', 'c']]   # 리스트 a의 ['a', 'b', 'c'] 리스트에서 'a' 값 인덱싱
>>> a[-1][0]                         # 리스트[리스트 인덱싱 숫자][리스트내 인덱싱 숫자]
'a'

# 3중 리스트 인덱싱
>>> a = [1, 2, ['a', 'b', ['Life', 'is']]]
>>> a[2][2][0]                       # 2번째 요소인 ['a', 'b', ['Life', 'is']]에서
'Life'                               # 2번째 요소인 ['Life', 'is']의 0번째 인덱싱
```


## 3. 리스트의 슬라이싱
```python
# 리스트의 슬라이싱 = 문자열과 동일, 마지막 값은 포함 하지 않음
>>> a = [1, 2, 3, 4, 5]
>>> b = a[:2]
>>> c = a[2:]
>>> b
[1, 2]
>>> c
[3, 4, 5]

# 중첩된 리스트에서 슬라이싱
>>> a = [1, 2, 3, ['a', 'b', 'c'], 4, 5]
>>> a[2:5]
[3, ['a', 'b', 'c'], 4]
>>> a[3][:2]                              # a의 3번째 ['a', 'b', 'c']의 첫번째부터 1번까지 
['a', 'b']
```


## 4. 리스트 연산: 더하기(+), 곱하기(*), 길이구하기(len함수)
```python
# 리스트 더하기 (+)
>>> a = [1, 2, 3]
>>> b = [4, 5, 6]
>>> a + b         
[1, 2, 3, 4, 5, 6]             # 문자열에서 "abc" + "def" = "abcdef"가 되는 것과 같은 이치

# 리스트 반복하기 (*)
>>> a = [1, 2, 3]
>>> a * 3
[1, 2, 3, 1, 2, 3, 1, 2, 3]    # 문자열에서 "abc" * 3 = "abcabcabc" 가 되는 것과 같은 이치

# 리스트 길이구하기_len함수
>>> a = [1, 2, 3]
>>> len(a)
3                              # len 함수는 문자열, 리스트외에도 튜플과 딕셔너리에도 사용가능

# 숫자와 문자열을 더하고 싶을 때
>>> a = [1, 2, 3]
>>> a[2] + "hi"                # 이는 정수와 문자는 더할 수 없기에 오류

>>>str(a[2]) + "hi"            # 정수나 실수를 문자열의 형태로 바꾸어 str() 내장 함수 사용
```


## 5. 리스트 수정과 삭제
* 수정은 단순히 해당 인덱스 설정 후 값 변경
* del 함수는 파이썬이 자체적으로 가지고 있는 삭제 함수로, del a[x]는 x번째 요소값을 삭제함을 뜻함
```python
# 리스트에서 값 수정하기
>>> a = [1, 2, 3]
>>> a[2] = 4                  # 단순하게 해당 인덱스 설정후, 값 변경
>>> a
[1, 2, 4]

# del 함수 사용해 리스트 요소 삭제하기
>>> a = [1, 2, 3]
>>> del a[1]
>>> a
[1, 3]

# 슬라이싱을 활용한 리스트 여러 요소 한번에 삭제
>>> a = [1, 2, 3, 4, 5]
>>> del a[2:]
>>> a
[1, 2]
```


## 6. 리스트 관련함수들
* 문자열과 마찬가지로 리스트 변수명 뒤에 '.'를 붙여서 리스트 관련 함수들을 사용할 수 있다. (ex. .count)
```python
# 리스트에 요소 추가(append)   "덧붙이다, 첨부하다"라는 뜻
>>> a = [1, 2, 3]
>>> a.append(4)               # .append(x)는 리스트의 맨 마지막에 x를 추가시키는 함수
>>> a
[1, 2, 3, 4]
>>> a.append([5,6])
>>> a
[1, 2, 3, 4, [5, 6]]          # 리스트에 다시 리스트를 추가한 결과

# 리스트 정렬(sort)
>>> a = [1, 4, 3, 2]
>>> a.sort()                  # .sort 함수는 리스트의 요소를 순서대로 정렬
>>> a
[1, 2, 3, 4]

>>> a = ['a', 'c', 'b']       # .sort 함수는 숫자와 알파벳 모두 순서대로 정렬한다
>>> a.sort()
>>> a
['a', 'b', 'c']

# 리스트 뒤집기(reverse)
>>> a = ['a', 'c', 'b']
>>> a.reverse()               # .reverse 함수는 리스트를 역순으로 뒤집어 준다
>>> a                         # 이때 리스트 요소를 순서대로 정렬한 후 역순 정렬이 아니라 
['b', 'c', 'a']               # 그저 현재의 리스트를 그대로 거꾸로 뒤집는다

# 위치 반환(index)
>>> a = [1,2,3]               # .index(x) 함수는 리스트에 x라는 값이 있으면 x의 위치값을 리턴
>>> a.index(3)
2
>>> a.index(0)                # 0은 a 리스트에 존재하지 않기 때문에 값 오류(ValueError) 발생
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
ValueError: 0 is not in list

# 리스트에 요소 삽입(insert)
>>> a = [1, 2, 3]
>>> a.insert(0, 4)            # .insert(a, b)는 리스트의 a번째 위치에 b를 삽입하는 함수
>>> a
[4, 1, 2, 3]                  # 파이썬에서는 숫자를 0부터 센다
>>> a.insert(3, 5)            # 리스트 a의 a[3], 즉 네 번째 요소 위치에 5라는 값을 삽입
>>> a
[4, 1, 2, 5, 3]

# 리스트 요소 제거(remove)
>>> a = [1, 2, 3, 1, 2, 3]
>>> a.remove(3)               # .remove(x)는 리스트에서 첫 번째로 나오는 x를 삭제하는 함수
>>> a
[1, 2, 1, 2, 3]               # a가 3이라는 값을 2개 가지고 있을 경우 첫 번째 3만 제거

>>> a.remove(3)
>>> a
[1, 2, 1, 2]                  # .remove(3)을 한 번 더 실행하면 다시 3이 삭제

# 리스트 요소 끄집어내기(pop)
>>> a = [1,2,3]            
>>> a.pop(1)                 # .pop(x)는 리스트의 x번째 요소를 돌려 주고 그 요소는 삭제
2
>>> a
[1, 3]

>>> a = [1,2,3]               # .pop() 공백이면 맨 마지막 요소를 보여주고 삭제
>>> a.pop()
3
>>> a
[1, 2]

# 리스트에 포함된 요소 x의 개수 세기(count)
>>> a = [1,2,3,1]
>>> a.count(1)               # .count(x)는 리스트 내에 x가 몇 개 있는지 보여주는 함수
2

# 리스트 확장(extend)
>>> a = [1,2,3]
>>> a.extend([4,5])          # .extend(x)에서 x에는 리스트만 올 수 있으며,
>>> a                        # 원래의 a 리스트에 x 리스트를 더하게 된다
[1, 2, 3, 4, 5]
>>> b = [6, 7]
>>> a.extend(b)
>>> a
[1, 2, 3, 4, 5, 6, 7]

#문자열의 join 함수 사용해 리스트 요소 붙이기(join)
>>> a = ['Life', 'is', 'too', 'short'] 
>>> ' '.join(a)
'Life is too short'
```
