맨날 구글링하다 더 이상 찾기 귀찮아서 쓰는 마크다운 문법들  



## 띄어쓰기와 단락바꾸기

- 띄어쓰기(space) 여러 번은 한번과 같다
- 엔터를 두번쳐야 단락이 바뀌고, 엔터 여러번 친다고 여러 줄이 띄어지는건 아님
- 강제로 여러 줄을 맘대로 띄고 싶다 하면 ```<br>```을 사용


## 제목(Header)

```markdown
# 헤더1
## 헤더2
### 헤더3
#### 헤더4
##### 헤더5
###### 헤더6
```

# 헤더1
## 헤더2
### 헤더3
#### 헤더4
##### 헤더5
###### 헤더6

<br>
<br>

## 목록(List)

1. 숫자 목록

```markdown
1. 하나
2. 둘
3. 셋
```

1. 하나
2. 둘
3. 셋
<br>
<br>

2. 기호 목록

```markdown
* 하나
    + 둘
        - 셋
```

* 하나
    + 둘
        - 셋
<br>
<br>

## 강조(Emphasis)

```markdown
_이탤릭_ , *이탤릭*
__볼드__ **볼드**
___이탤릭볼드___
***이탤릭볼드***
**이렇게 _할수도_**
~~취소선~~
```

    _이탤릭_ , *이탤릭*
    __볼드__ **볼드**
    ___이탤릭볼드___
    ***이탤릭볼드***
    **이렇게 _할수도_**
    ~~취소선~~
